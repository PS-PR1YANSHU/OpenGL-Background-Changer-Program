# OpenGL-Background-Changer-Program
A Background Changer Program made using glfw & glew in OpenGL(C++)

## About
This is my First OpenGL Program which Changes the BackGround Color based on the Key You Press. Source Code is also Available

## KEYS
- R → Red
- G → Green
- B → Blue
- 1 → Remove Red
- 2 → Remove Green
- 3 → Remove Blue
- Backspace → Remove All Color
- Enter / Space → Enable RGB to FULL
- L → Light Gray
- O → Orange
